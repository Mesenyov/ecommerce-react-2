# TechStore — Ecommerce (React) | Итерация 2

Вторая итерация проекта интернет-магазина техники. 

В данной версии фокус смещен со статической верстки на **управление состоянием приложения** и создание интерактивности. Реализован полноценный каталог товаров с переключением категорий, фильтрацией, сортировкой и глобальной корзиной.

## Что реализовано (Итерация 2)
*   **Глобальное состояние (Container):** 
    *   стейт приложения (`pageType` и `cart`) поднят на уровень корневого компонента (`App.jsx`);
    *   данные передаются между несвязанными компонентами
*   **Интерактивный каталог:**
    *   переключение между категориями (TV, Phone, Laptop) с рендерингом соответствующих товаров;
    *   универсальный компонент `CategoryListing`.
*   **Сортировка и Фильтрация:**
    *   управление локальным состоянием формы фильтров;
    *   динамическое формирование списка брендов на основе выбранной категории;
    *   фильтрация по бренду и диапазону цен с применением по кнопке "Apply";
    *   сортировка списка товаров по цене (Low to High / High to Low).
*   **Полноценная корзина (Cart):**
    *   отдельная страница корзины;
    *   возможность изменения количества товаров и их полного удаления;
    *   синхронизация количества товаров в корзине со счетчиками на карточках в каталоге;
    *   функционал очистки корзины.

*(Все визуальные фичи первой итерации: адаптивность, Sticky Header, карусель изображений сохранены). Список брендов выводится :)*

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Mesenyov/ecommerce-react-2.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd ecommerce-react-2
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```
Откройте в браузере `http://localhost:5173`
